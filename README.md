📊 ETL-пайплайн мониторинга цен товаров

Проект реализует автоматический сбор, хранение и визуализацию цен товаров с использованием PostgreSQL, Docker и Streamlit.

Проект развернут на VPS (Ubuntu).

Дашборд доступен по адресу:

http://46.149.77.76:8502

Все сервисы работают в Docker-контейнерах:

- PostgreSQL — хранит данные

- ETL-скрипт — качает товары и пишет в БД

- Streamlit — показывает дашборд

🔄 Как работает проект

ETL-скрипт получает товары из API
↓
Цена немного изменяется случайным образом
↓
Данные сохраняются в PostgreSQL
↓
Старые записи автоматически удаляются
↓
Streamlit получает данные из базы
↓
На странице отображаются таблицы и графики

⚙️ Что реализовано
Получение данных из внешнего API   
Автоматическая загрузка товаров в PostgreSQL   
ETL-процесс с периодическим обновлением данных   
Хранение истории изменения цен 
Ограничение истории цен (100 записей на товар)   
Визуализация данных в Streamlit  
Docker-контейнеризация проекта  
Сохранение данных в Docker volume
Автоматический перезапуск контейнеров после reboot сервера

📦 Источник данных

Используется тестовое API товаров:   

DummyJSON Products API   

Загружаются:   

название товара   
категория  
цена   

Для демонстрации динамики цен при каждом обновлении применяется небольшое случайное изменение стоимости.   

📈 Содержание дашборда   
Latest Products - Таблица с последними актуальными ценами товаров.   

Average Price by Category - Средняя цена товаров по категориям.   

Price History - График изменения цены выбранного товара во времени.
